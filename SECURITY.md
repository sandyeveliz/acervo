# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.1.x   | ✅        |

## Reporting a Vulnerability

Please do not report security vulnerabilities through public GitHub issues.

Instead, use GitHub's private vulnerability reporting:
**Security → Report a vulnerability** in this repository.

You can also reach the maintainer directly via GitHub:
[@sandyeveliz](https://github.com/sandyeveliz)

We will respond within 7 days and work with you to resolve
the issue before any public disclosure.

## Scope

Issues considered in scope:
- Data leakage between user contexts
- Graph corruption via malicious input
- Dependency vulnerabilities

Thank you for helping keep Acervo safe.
